# 戦闘ルール

戦闘は、スクエア状のマップ上で行われます。
各キャラクターはマスに配置され、移動や攻撃を行います。

本章では、戦闘の進行手順と、戦闘中に使用される基本的なルールを説明します。

## 22.1 戦闘の基本

戦闘は、以下の要素によって構成されます。

- ラウンド
- ターン
- 行動

## 22.2 戦闘開始

戦闘が開始されると、以下の手順を行います。

1. 戦闘に参加するキャラクターを決定する
2. 各キャラクターをマップ上に配置する
3. ラウンドを開始する

## 22.3 ラウンドとターン

戦闘はラウンド単位で進行します。

ひとつのラウンドは、以下のサイクルで構成されます。

1. イニシアチブプロセス
2. 各キャラクターのターン
3. (未行動のキャラクターがいない場合)ラウンドの終了

## 22.4 イニシアチブプロセス

ラウンド開始時、イニシアチブプロセスを行います。

イニシアチブプロセスでは、次に行動するキャラクターを決定します。

### 手順

1. 行動可能なキャラクターを確認する
2. 行動するキャラクターを1体選ぶ
3. 選ばれたキャラクターはターンを開始する

## 22.5 行動の種類

キャラクターは、自身のターン中に以下の行動を行うことができます。

### 行動構成

- ムーブアクション（移動）
- メジャーアクション（主行動）

## 22.6 移動

ムーブアクションでは、キャラクターはマップ上を移動できます。

- 移動距離は「すばやさ」によって決定されます
- 移動は縦横方向に行います

## 22.7 攻撃とダメージ

### 攻撃の手順

1. 使用するわざを宣言する
2. 命中判定を行う
3. 成功度を算出する
4. ダメージを算出する

### 命中判定

- 攻撃は、原則として攻撃側のみが行為判定を行います
- 対象は回避判定を行いません

技の種別によって命中判定に用いる能力値が決定します。

|種別|能力値|
|--|--|
|ぶつり|こうげき|
|とくしゅ|とくこう|
|へんか|（わざタグによる）|

成功度が1以上であれば攻撃は命中したものとし、ダメージ算出を行います。

### ダメージ算出
```
ダメージ ＝（威力＋成功度）d6
```
- 威力は、わざの基本ダイス数です
- 成功度が高いほど、振るダイスは増えます
- とくちょう、とくせい、わざタグの影響でさらなる補正がかかる場合があります

## 22.8.1 防御とダメージ軽減

ダメージが算出された後、対象は防御による軽減を行います。

- ぶつりわざ：ぼうぎょを使用
- とくしゅわざ：とくぼうを使用
```
算出ダメージ ＝ ダメージ − 防御値
```

算出ダメージは 0 未満になりません。(最低で1となります)

## 22.8.2 タイプ相性

防御による軽減を適用後、タイプ相性による補正を行います。わざのタイプが、攻撃を受けたキャラクターの弱点や耐性であった場合、算出ダメージに修正を行います。

|相性|処理|
|--|--|
|弱点（+4ｄ）|4ｄを振り、ダメージに加えます|
|弱点（+2ｄ）|2ｄを振り、ダメージに加えます|
|耐性（1/2）|ダメージを半分にします（端数切り捨て）|
|耐性（1/4）|ダメージを4分の1にします（端数切り捨て）|
|耐性（1/10）|ダメージを10分の1にします（端数切り捨て）|

また、耐性による修正によってダメージは 0 未満になりません。(最低で1となります)  
このように修正を加えたダメージを最終ダメージとして、対象となったキャラクターのHPから減少させます。


## 22.9 戦闘の終了

下のいずれかを満たした場合、戦闘は終了します。

- 敵対する勢力のいずれかが戦闘不能になった
- 戦闘の継続が不可能、または不要になった
- ゲームマスターが戦闘終了を宣言した
